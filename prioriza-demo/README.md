# PRIORIZA Comercial — demonstração sintética v0.2

Esta pasta contém uma demonstração pública e isolada do fluxo **priorização → preparação de Discovery**.

## Regra de segurança

- somente dados públicos, sintéticos ou anonimizados;
- nenhum dado de cliente, contato, pipeline, proposta ou nota interna;
- nenhuma credencial ou informação confidencial;
- nenhuma ação externa automática;
- toda saída exige revisão humana.

## Escopo da demonstração

1. priorização de um prospect sintético por critérios transparentes;
2. envio de um prospect elegível para preparação de Discovery;
3. registro de fatos e fontes;
4. separação entre fatos confirmados, observações sem fonte e hipóteses;
5. geração de roteiro com abertura, agenda, SPIN, BBE, objeções e próximo compromisso.

O produto canônico permanece no repositório privado `nandokonzen-code/prioriza-comercial`. O repositório público `Atlas` continua sendo tratado como legado/demonstração histórica; esta pasta existe somente para validação pública segura da experiência.
